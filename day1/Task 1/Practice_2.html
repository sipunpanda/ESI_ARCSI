HTML stands for HyperTextMarkupLanguage. It is used to display the document in the web browser. HTML pages can be developed to be simple text or to be complex multimedia program containing sound, moving images and java applets. HTML is considered to be the global publishing format for Internet. It is not a programming language. HTML was developed by Tim Berners-Lee. HTML standards are created by a group of interested organisations called W3C(World Wide Web Consortium). In HTML formatting is specified by using tags.A tag is a format name surrounded by angle brackets. End tags which switch a format off also contain a forward slash.
